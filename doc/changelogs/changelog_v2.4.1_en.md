# v2.4.1

## Bugfixes
- **HTTPS**: Due to changes in v2.4.0, reconnecting via HTTPS on Android smartphones caused issues. This has now been fixed.
- **Version check**: Checking for the latest Kittyhack version now works immediately after restarting the cat flap.

## Improvements
- **Loading speed**: The loading time of the Kittyhack web interface has been significantly reduced.
- **DB/Config restore**: Additional validations and a confirmation dialog were added for restoring the database or configuration.
