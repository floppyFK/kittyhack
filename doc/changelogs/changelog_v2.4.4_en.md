# v2.4.4

Just another small bugfix release following the major 2.4.0 release. The changes for v2.4.0 can be found [here](https://github.com/floppyFK/kittyhack/blob/main/doc/changelogs/changelog_v2.4.0_en.md).

## Bugfixes
- **Mozilla browser support**:
	- The sporadic flickering of images in Live View as well as in Event View has been fixed (mobile and desktop).
	- Navigation from the Kittyhack WebGUI to external pages now works correctly in the Android version of Mozilla as well.
